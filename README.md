## Catering Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/108a42d8ffc0497294ced27db30ec315/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Employee, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Dishes
5. Forum
6. Announcement Information
7. Hotel Introduction
8. Contact Us
9. Personal Center
10. Dish Favorites
11. Dish Orders
12. Backend Management



**Employee Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Dish Management, Dish Review Management, Dish Favorite Management, Dish Order Management
6. Forum Management
7. Supplier Management
8. Announcement Information Management



**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Administrator Management
4. Dish Management, Dish Review Management, Dish Favorite Management, Dish Order Management
5. Basic Data Management (Dish Type Management, Item Type Management, Member Level Type Management, Announcement Type Management, Single Page Data Type Management)
6. Forum Management
7. Supplier Management
8. Announcement Information Management
9. Single Page Data Management
10. User Management
11. Employee Management
12. Carousel Information
